# SunXCorr

Fast solar image cross-correlation and alignment library.

## Installation

Clone the repository and install with pip:

```
pip install .
```

## Requirements
- numpy
- scipy
- numba
- astropy
- sunpy
- matplotlib

## Usage

Import the main routines from the `sunxcorr` package.

## Optional Test Data

Large FITS files for testing are not included by default.
To download them, run:

```
python download_test_data.py
```

Or download manually from your preferred source and place them in the `fits_files/` directory.
