sunxcorr.worker_setup

Persistent worker helpers used by SunXCorr.

This module creates manager-backed shared structures and spawns persistent worker processes that execute correlation workloads. Workers are intended to be started once (at the root orchestrator) and reused across many optimization calls to avoid process spawn overhead.

Examples

>>> from sunxcorr.worker_setup import setup_persistent_workers
>>> callable(setup_persistent_workers)
True
