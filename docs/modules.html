API Reference

sunxcorr

SunXCorr package.

sunxcorr.optimization

Optimization functions for coalignment (shift and/or scale search).

sunxcorr.orchestrator

Orchestrator node for managing coalignment processes.

sunxcorr.correlation_jit

High-performance correlation helpers (Numba/JIT).

sunxcorr.worker_setup

Persistent worker helpers used by SunXCorr.

sunxcorr.utils

Utility functions for coalignment processes.

sunxcorr.process_result

Data structures for coalignment process results.

sunxcorr.single_map_process

Single-map cross-correlation process (shift-only optimization).

sunxcorr.synthetic_raster_process

Synthetic raster generation and optimization process.
