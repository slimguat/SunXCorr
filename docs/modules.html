sunxcorr.base_node

Base node functionality for the SunXCorr coalignment tree.

This module defines the abstract CoalignmentNode base class used to build tree-structured coalignment workflows. Nodes may be composite (orchestrator) or leaf (processing step). The class provides resource lookup helpers (worker pool, queues), tree management helpers, and result collection utilities.

Typical usage:

root = Orchestrator(n_workers=4) process = SomeLeafProcess() root.add_child(process) root.execute()

Examples

>>> from sunxcorr.base_node import CoalignmentNode
>>> # The base class is abstract; constructing a subclass or using
>>> # `Orchestrator` / concrete processes is the typical workflow.
